# AlgoStudio

A language-agnostic DSA (Data Structures & Algorithms) project: implementations and a web-based visualizer.

## Structure

- **cpp/** — C++ algorithm implementations (CMake, C++20)
- **apps/visualizer/** — Web app for visualizing algorithms (React + TypeScript + Vite)
- **docs/** — Roadmap and planning (e.g. [DSA Study Roadmap](docs/roadmap.md))

## Quick start

- **C++**: From `cpp/`, run `cmake -B build && cmake --build build`
- **Visualizer**: From `apps/visualizer/`, run `npm install && npm run dev`
